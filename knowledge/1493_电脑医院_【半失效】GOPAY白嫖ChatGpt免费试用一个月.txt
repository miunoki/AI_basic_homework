【标题】【半失效】GOPAY白嫖ChatGpt免费试用一个月

【帖子正文】
今天Openai好像把域名邮箱的免费试用给限制了。经过L站大佬研究，google邮箱和outlook邮箱都还有免费试用。楼主是日本节点，能刷出来免费试用。下面是焚诀订阅free plus教程，把openai给耗秃。（注意，gopay订阅似乎不是很稳定，估计可能几天就会掉，也有可能用很多天。不要多个gopay绑定多个账户，最好一号一绑，最为稳妥。）

5.9号更新，+86账号目前支付会遇到错误，此教程前几步需要换成印度尼西亚的手机号注册gopay。
写在前边，注意，最好先检测一下有没有资格。此攻略有点长，耗时有点长，别到头来没有free plus资格。
1.首先手机谷歌商店下载WhatsApp和gopay。
2.注册WhatsApp账户，+86手机号就可以注册。
3.注册gopay账户，用WhatsApp接码，完成注册后，设置pin以及邮箱（邮箱可能不需要），获取1lp。
以上支付方式注册完成，下午gopay的检测可能比较严格，注册会有各种各样的问题，最好早上或者晚上再看看（注册不成功的）。

4.获取日本家庭节点。楼主使用的是1024proxy，新注册的用户可以买1个0.99dollar1Gb的动态ip家庭住宅，只用来薅羊毛的话根本用不完。以下是我的邀请连接，随缘点击。https://api.1024proxy.com/share/hkjdfzb1d
5.配置指纹浏览器以及获取代理地址。
指纹浏览器楼主用的是adspower，其它指纹浏览器自测，配置都差不多。
先去1024上获取代理配置地址。
https://file.cc98.org/v4-upload/d/2026/0508/sthk5f5j.webp
打开adspower，添加代理，如下图。

https://file.cc98.org/v4-upload/d/2026/0508/mq35kxas.webp
显示成功以后，确认。如果显示获取失败可能是当前ip未在白名单里面，你需要返回1024官网，把本机的代理ip就是挂vpn后显示的ip加入到白名单里面，如下。
https://file.cc98.org/v4-upload/d/2026/0508/q2nzbxmm.webp
具体ip怎么看，可以直接搜ping0.cc，里面的就是你的代理ip。

7.打开chatgpt官网，登录账户，右上角会有一个获取plus。或者是以下链接。
https://chatgpt.com/?promo_campaign=plus-1-month-free#pricing
https://file.cc98.org/v4-upload/d/2026/0508/0bbcw4zx.webp
显示0元试用就是有资格。

8.在零元试用界面，f12打开控制台，输入以下代码提取stripe连接。这一步是避免切换到印度尼西亚，不让你用gopay，显示支付取消的。
(async function generatePlusHostedLink() {

console.log("⏳ 正在获取 Session Token...");

// ── 1. 获取当前登录的 Access Token ──────────────────────────────────────

let accessToken;

try {

const session = await fetch("/api/auth/session").then((r) => r.json());

accessToken = session?.accessToken;

if (!accessToken) throw new Error("accessToken 为空");

} catch (e) {

console.error("❌ 获取 Token 失败，请确保已登录 ChatGPT：", e.message);

return;

}

console.log("✅ Token 获取成功");

// ── 2. 构造请求 Payload ──────────────────────────────────────────────────

// plan_name : chatgptplusplan（Plus 个人计划）

// promo_campaign_id : plus-1-month-free（首月免费优惠活动）

// checkout_ui_mode : hosted（关键！让 OpenAI 返回 Stripe Hosted URL）

// billing_details : 填写账号所在地区，影响币种和税率

const payload = {

plan_name: "chatgptplusplan",

billing_details: {

country: "ID", // 欧洲账号用 DE/EUR；美国账号改 US/USD

currency: "IDR", // 欧洲账号用 EUR；美国账号改 USD

},

cancel_url: "https://chatgpt.com/#pricing",

promo_campaign: {

promo_campaign_id: "plus-1-month-free",

is_coupon_from_query_param: false,

},

checkout_ui_mode: "hosted",

};

// ── 3. 发送请求 ──────────────────────────────────────────────────────────

console.log("⏳ 正在请求 Stripe 长链接...");

let data;

try {

const response = await fetch(

"https://chatgpt.com/backend-api/payments/checkout",

{

method: "POST",

headers: {

Authorization: `Bearer ${accessToken}`,

"Content-Type": "application/json",

},

body: JSON.stringify(payload),

}

);

data = await response.json();

if (!response.ok) {

console.error("❌ 请求失败，HTTP", response.status, data);

return;

}

} catch (e) {

console.error("❌ 网络请求异常：", e.message);

return;

}

// ── 4. 输出结果 ──────────────────────────────────────────────────────────

const hostedUrl = data?.url || data?.stripe_hosted_url || data?.checkout_url;

if (!hostedUrl) {

console.warn("⚠️ 未找到长链接，原始响应如下：");

console.log(data);

return;

}

console.log("─".repeat(60));

console.log("✅ 生成成功！");

console.log("");

console.log("📋 Checkout Session ID :", data.checkout_session_id);

console.log("🏢 Processor Entity :", data.processor_entity);

console.log("💰 Plan : ChatGPT Plus（首月 €0/免费试用）");

console.log("");

console.log("🔗 Stripe 长链接：");

console.log(hostedUrl);

console.log("─".repeat(60));

})();
