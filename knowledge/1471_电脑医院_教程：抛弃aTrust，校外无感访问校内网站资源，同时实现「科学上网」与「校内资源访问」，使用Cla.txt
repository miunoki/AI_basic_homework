## 写在前面

### 使用场景

暑假在家，在校外的时候，我们经常需要访问校内服务器和资源，同时也需要使用代理访问国际学术网站，传统的方案一般是是使用 RVPN 客户端，但与现有代理冲突，需要来回切换，非常繁琐，并且很多情况下关掉代理，会导致某些网站检测到你的真实IP，到时账号被ban（比如Claude、Gemini这些不对大陆提供服务的网站）

本教程提供一个**优雅的解决方案**：通过 `zju-connect` 和 `Clash Verge`，实现**智能分流**，让校内外资源访问无缝切换。

### 最终效果

最终效果差不多是这样的

```
开机后自动：
```
- **1**：zju-connect 连接到校内 VPNClash Verge 根据规则自动分流
- **2**：访问 CC98、教务系统等 → 走 zju-connect → 校内网络
- **3**：访问 Google、GitHub、Claude 等 → 走国际代理
- **4**：个人网站、国内网站 → 直连

### 整体架构

```
你的应用程序（浏览器/终端等）
 ↓
Clash Verge (系统代理)
 ↓ (根据规则分流)
 ├─→ zju-connect (127.0.0.1:21080)
 │ ↓
 │ 浙大 VPN 服务器
 │ ↓
 │ 校内资源 (*.zju.edu.cn, 10.x.x.x)
 │
 └─→ 其他代理节点
 ↓
 国际网站 (Google/GitHub/Claude)
```

- macOS 10.15+（我用的是macos），但是其他操作系统整体思路一致。
- 浙大统一身份认证账号

1. **zju-connect** - 校内 VPN 客户端
 
 - 下载地址: (https://github.com/Mythologyli/zju-connect/actions/workflows/build.yml?query=branch%3Aatrust)

 - 下载地址: https://github.com/clash-verge-rev/clash-verge-rev/releases
 - 其他 Clash 客户端也可以，配置方法类似
3. **代理订阅**
 
 - 任何支持 Clash 的机场订阅
 - 或自建代理服务器

## 安装配置 zju-connect

### 第一步：下载 zju-connect

1. 访问 (https://github.com/Mythologyli/zju-connect/actions/workflows/build.yml?query=branch%3Aatrust)
2. 选择 `atrust` 分支的最新构建
3. 下载对应版本：
 
 - **Mac M 系列芯片**：(https://github.com/Mythologyli/zju-connect/actions/runs/18283178062/artifacts/4233238125)
 - **Mac Intel 芯片**：`(https://github.com/Mythologyli/zju-connect/actions/runs/18283178062/artifacts/4233238192)
 - **WinX86**：(https://github.com/Mythologyli/zju-connect/actions/runs/18283178062/artifacts/4233238085)
4. 解压并移动到工作目录：

```bash
cd ~/zju-connect
```

### 第二步：下载验证码处理脚本

创建 `~/zju-connect/tmp/point_picker.py`，有两个版本可选：

#### 版本 A：带图形界面（需要 tkinter 和 PIL）

point_picker.py下载链接：
(https://alist.x0bang.com/d/LocalDisk/%E6%96%87%E4%BB%B6%E5%88%86%E4%BA%AB/98/point\_picker.py?sign=j2CNRWJPluB5D8MTBjPcGZbQRtdvUyshbJ9AwNaoBac=:0)

```python
import tkinter as tk
from PIL import Image, ImageTk
from io import BytesIO

def pick_points(image_buffer, num_points=3, display_width=800):
 img = Image.open(BytesIO(image_buffer))
 orig_width, orig_height = img.width, img.height

 scale = orig_width / display_width
 new_size = (int(orig_width / scale), int(orig_height / scale))
 img = img.resize(new_size)

 root = tk.Tk()
 tk_img = ImageTk.PhotoImage(img)
 canvas = tk.Canvas(root, width=img.width, height=img.height)
 canvas.pack()
 canvas.create_image(0, 0, anchor="nw", image=tk_img)

 coordinates = []

 def on_click(event):
 x = int(event.x * scale)
 y = int(event.y * scale)
 canvas.create_oval(
 event.x - 5, event.y - 5, event.x + 5, event.y + 5, fill="red"
 )
 coordinates.append()
 print()
 if len(coordinates) >= num_points:
 root.after(1000, root.destroy)

 canvas.bind("<Button-1>", on_click)
 root.mainloop()

 return {"coordinates": coordinates, "width": orig_width, "height": orig_height}

if __name__ == "__main__":
 with open("../CheckCode.jpg", "rb") as f:
 image_buffer = f.read()
 result = pick_points(image_buffer, num_points=3)
 import json
 print(json.dumps(result, separators=(",", ":")))
```

**版本 B 使用方法：**

2. 在预览中使用「工具」→「矩形选框」

### 第三步：创建配置文件

配置文件下载链接：
(https://alist.x0bang.com/d/LocalDisk/%E6%96%87%E4%BB%B6%E5%88%86%E4%BA%AB/98/config.toml?sign=KZDpYmLyfK2KLT01jPzRLmW2-J8J\_kEjAJFhZZBaw90=:0)

创建 `~/zju-connect/tmp/config.toml`：

```toml
username = "你的学号"
password = '你的VPN密码'
protocol = "atrust"
graph_code_file = "CheckCode.jpg"
client_data_file = "clientData.json"

# Advanced settings
server_address = "vpn.zju.edu.cn"
server_port = 443
disable_server_config = false
disable_zju_config = false
disable_zju_dns = false
disable_multi_line = false
proxy_all = false
socks_bind = ":21080"
socks_user = ""
socks_passwd = ""
http_bind = ":28888"
shadowsocks_url = ""
dial_direct_proxy = ""
tun_mode = false
add_route = false
dns_ttl = 3600
disable_keep_alive = false
zju_dns_server = "10.10.0.21"
secondary_dns_server = "114.114.114.114"
dns_server_bind = ""
dns_hijack = false
debug_dump = false

port_forwarding = []
custom_dns = []
custom_proxy_domain = []
```

**注意事项：**

- 密码如果包含特殊字符（如 `!`），建议使用单引号包裹：`password = '密码'`
- SOCKS5 代理端口为 `21080`，HTTP 代理端口为 `28888`

### 第四步：首次运行

```bash
cd ~/zju-connect
./zju-connect -config tmp/config.toml
```

**操作流程：**

1. zju-connect 会下载验证码图片 `CheckCode.jpg`
2. 看到 `Please enter the graph check code JSON:` 提示后
3. 迅速打开一个**新终端**
4. 在**新终端**快速运行，必须要快速完成，在 1-2 分钟内就会失效的：
 ```bash
 cd ~/zju-connect/tmp
 python point_picker.py
 ```
5. 在弹出的窗口中点击验证码的 3 个目标位置
6. 终端会输出 JSON 格式的坐标，**立即复制**整行
7. 粘贴到运行 zju-connect 的终端，按回车
8. 输入手机收到的短信验证码
9. 看到 `KeepAlive using TCP: OK` 表示连接成功

**重要提示：**

- 验证码有效期很短，操作要快！
- 建议提前打开两个终端窗口准备好命令

### 第五步：验证连接

```bash
# 测试 SOCKS5 代理
curl -x socks5://127.0.0.1:21080 http://cc98.org

# 测试 HTTP 代理
curl -x http://127.0.0.1:28888 http://cc98.org
```

如果返回 CC98 的 HTML 内容或 301 重定向，说明连接成功！
![](https://file.cc98.org/v4-upload/d/2025/1215/pikjl2iw.webp)

**首次登录成功后，会在 `~/zju-connect` 目录下生成 `clientData.json`，下次运行时会自动登录，无需再输入验证码！**

### 第六步：测试自动重连

```bash
# 按 Ctrl+C 停止 zju-connect
# 然后重新启动
cd ~/zju-connect
./zju-connect -config tmp/config.toml
```

应该看到：

```
Using saved client data
Login successful!
SOCKS5 proxy listening on 127.0.0.1:21080
KeepAlive using TCP: OK
```

自动登录成功！
